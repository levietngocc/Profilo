⟡ SELF-LEARNING PROJECT / FRONTEND

TY FOODS WEB ORDER

QR Table Ordering · Customizable Desserts · PWA‑like Experience

🎯 Role & Objective

Role: Full-stack Developer (Self-learning Project)
Goal: Build a complete digital ordering system for a dessert shop – customers scan QR code, select items with full customization (size, sweetness, temperature, toppings), add to cart, and send order directly to kitchen. The system also includes feedback/rating and bilingual interface (EN/VI).

Inspired by real‑world needs of small F&B businesses, this project demonstrates a production‑ready frontend with zero backend cost (uses localStorage for data persistence).

⚙️ Skills gained

  • Frontend Architecture – Designed a fully responsive PWA‑style web app with multi‑view navigation (Home / Menu / Cart / Success).
  • Dynamic Product Customization – Implemented real‑time price calculation for size options, topping selection, and quantity.
  • LocalStorage & State Management – Managed cart, order history, and pending feedback without a backend – perfect for offline demo and small shops.
  • Bilingual UI (EN/VI) – Built a complete language toggle with all UI text and product descriptions translated.
  • QR Table Order Flow – Simulated “order at table” experience: guest enters name/table, adds items, sends order to kitchen (simulated).
  • Feedback & Rating System – Allowed customers to rate past orders, select specific items, and submit detailed feedback.

✨ Key features

  • 📱 Mobile‑first design, smooth scrolling, keyboard‑aware layout
  • 🍨 30+ products across 4 categories (Base, Special, Snacks, Others)
  • 🎨 Customization: size (S/M/L), sweetness (4 levels), temperature (ice/hot), 27+ toppings
  • 🧾 Cart review with detailed item breakdown, kitchen note field
  • 📋 Order confirmation modal before sending to kitchen
  • ⭐ Post‑order rating: choose order from history, rate each item, submit issues & notes
  • 🌍 Full English/Vietnamese toggle with persistent language preference
  • 🖼️ Twemoji integration for consistent cross‑platform emoji display
